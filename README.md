# Student Portfolio Website

## Overview

This project was developed as part of the **COS 106 – Introduction to Web Technologies** course. It is a responsive multi-page website designed to serve as a personal student portfolio and academic management platform. The website showcases personal information, academic background, skills, projects, and provides a contact page for visitors.

The project demonstrates the use of **HTML5, CSS3, and JavaScript** to create a modern, responsive, and interactive website.

---

## Features

- Responsive design for desktop, tablet, and mobile devices
- Multi-page website
- Responsive navigation menu
- Hero section
- About Me section
- Skills section
- Academic information
- Portfolio/Projects section
- Animated statistics/counters
- Contact form
- Professional footer
- Clean and user-friendly interface
